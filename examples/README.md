## Installing on Windows

- mingw-w64

   ```
   pacman -S mingw-w64-x86_64-glfw
   ```

## Installing on Linux

- Debian

   ```
   sudo apt-get install liballegro5-dev
   ```

   ```
   sudo apt-get install libglfw3-dev
   ```

   ```
   sudo apt-get install libsdl2-dev
   ```
